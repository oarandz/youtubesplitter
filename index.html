Video Wall

multiview gallery · main + thumbnails · live wires
Click any channel (or press its number / the arrow keys) to make it the big main window — the rest tile as thumbnails. Give each line a direct live link for instant, reliable playback, e.g. Sky News | https://youtube.com/watch?v=…. To auto-follow a channel's current live stream (so it picks up the next show when they restart, and catches a channel like Number 10 the moment it goes live), use Name | channel:UCxxxx — but this needs a free YouTube Data API key pasted into YT_API_KEY at the top of the file, because YouTube no longer lets a browser find the live video without one. No key = use direct links. Drag the dividers between windows to resize them. The wires bar merges all feeds, tagged by source, newest first, refreshing every 30 seconds: new headlines flash orange for 30s, anything timestamped within 5 minutes flashes red. Feeds use public CORS proxies (one per feed).
